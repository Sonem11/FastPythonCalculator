# Fast Python Calculator

A simple and fast calculator written in Python.  
It supports basic operations: addition, subtraction, multiplication, and division.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/), with error handling for division by zero
